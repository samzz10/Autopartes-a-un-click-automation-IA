# Autopartes-a-un-click-automation-IA

Sistema de automatización empresarial para AUTOPARTESAUNCLICK usando Python, APIs e inteligencia artificial.

## Objetivo
Automatizar procesos repetitivos de la empresa como:

- Envío automático de cotizaciones
- Chatbot para clientes
- Automatización de correos

## Tecnologías
- Python
- APIs
- Automatización de procesos
- Inteligencia Artificial
- Bases de datos o exel
